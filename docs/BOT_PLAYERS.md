# Bot Players

Shared password:
- Set `SYSTEM_BOT_PASSWORD` before starting `game-backend`.
- There is no committed default password.
- Use a local-only placeholder for development and a strong server-only value for production.

Login details:
- Sign in with either the username or the email shown below.
- Bot accounts use the server-provided `SYSTEM_BOT_PASSWORD`.
- Usernames are locked after creation and cannot be changed.
